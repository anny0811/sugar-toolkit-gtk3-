# sugar-toolkit-gtk3-
Sugar Toolkit GTK3 provides GTK3-based UI components and helpers used by Sugar activities and the Sugar desktop environment.

## Overview
This repository contains the GTK3 implementation of the Sugar toolkit. It is used by Sugar activities to provide consistent user interface elements such as toolbars, palettes, alerts, and presence-related helpers.

## Modules
- **sugar3.graphics**
  UI widgets such as toolbars, palettes, icons, and alerts.

- **sugar3.activity**
  Base classes and helpers for writing Sugar activities.

- **sugar3.presence**
  Interfaces for collaboration and presence services.
